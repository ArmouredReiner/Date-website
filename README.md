# Will You Be My Valentine? 💖

### A Playful Way to Ask Someone Special

---

## What is this?

A super cute, interactive Valentine's Day card that's basically the most adorable way to ask someone to be your Valentine! Featuring a pixel art character, floating hearts, sparkly animations, and the iconic Mario theme to set the mood! 🎮💕

---

## Features

### Interactive Experience
- **The "NO" Button** - Tries to escape every time you hover over it! 😱
- **The "YES" Button** - Triggers an epic victory celebration! 🏆
- **Toggle Music Button** - Press play to hear the nostalgic Super Mario Bros. theme

### Visual Magic
- Soft pink gradient background with floating orbs ✨
- Fluffy animated clouds drifting by ☁️
- Twinkling sparkle stars across the screen ⭐
- Flowing grass at the bottom 🌿
- Retro scanline effect for that classic gaming vibe 📺

### The Big Moment
When your special someone clicks **YES**:
- 🎵 Victory music starts playing
- 💖 Floating hearts rain down everywhere
- 🎊 Confetti bursts in celebration
- 🎂 A cute pixel cake appears!
- 😄 The pixel character does a happy bounce

---

## Tech Stack

- **HTML5** - Structure & content
- **CSS3** - All the pretty styling, animations, and pixel art magic
- **JavaScript** - Interactive logic and audio control
- **HTML5 Audio** - For playing the actual Mario music files

### Fonts
- *Press Start 2P* - That classic 8-bit gaming font

### Audio
- **Background Music:** Super Mario Bros. Theme
- **Victory Music:** Super Mario Bros. Level Complete

---

## How It Works

1. Open `index.html` in any web browser
2. Click the **♫ OFF** button to start the Mario theme music
3. Watch the pixel character's sad face as you hover over "NO" 😢
4. Finally click **YES** and watch the magic happen! 🎉

---

## Customization Ideas

Want to make it your own? Here are some fun ideas:

- 🎨 Change the color scheme in the CSS gradient variables
- 💬 Edit the "NO" button messages in the `noMessages` array
- 🖼️ Replace the pixel character with your own design
- 🎵 Swap out the music files for different songs
- 📱 Add more animations or effects
